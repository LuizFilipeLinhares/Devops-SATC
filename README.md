# 🚀 Landing Page SaaS Moderna

Uma landing page moderna desenvolvida com React + TypeScript, com design premium inspirado em startups de tecnologia.

## Interface elegante com:

- 🎨 Gradientes modernos
- ✨ Glassmorphism
- 💎 Cards interativos
- 🚀 Botões animados
- 📱 Layout responsivo
- 🖼 Preview visual profissional

Landing page estilo SaaS com Hero impactante, seções de benefícios, pricing e call-to-action.

## 🛠 Tecnologias Utilizadas

- React
- TypeScript
- Vite
- CSS moderno (Glass + Gradients)
- Design Responsivo

## ⚙️ Como Rodar o Projeto

### 1️⃣ Criar o projeto com Vite
```bash
npm create vite@latest landing-page -- --template react-ts
```

### 2️⃣ Instalar dependências
```bash
cd landing-page
npm install
```

### 3️⃣ Rodar em ambiente de desenvolvimento
```bash
npm run dev
```

O projeto estará disponível em:
```
http://localhost:5173
```
