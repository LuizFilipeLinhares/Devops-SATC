# DevOps vs SRE

## 🔧 DevOps

**DevOps é uma cultura de trabalho.**  
Une Desenvolvimento (Dev) e Operações (Ops) para entregar software:

- Mais rápido 🚀  
- Com mais qualidade ✅  
- Com menos erros 💥  

**Foco principal:** automação e colaboração  
Exemplos: CI/CD, containers, cloud, infraestrutura como código.

**Resumo:**  
> DevOps é a forma moderna de desenvolver e publicar sistemas em produção.

---

## 📊 SRE (Site Reliability Engineering)

**SRE é um cargo/especialização.**

**Foco principal:** confiabilidade e estabilidade dos sistemas.

Trabalha com métricas como:
- SLA
- SLO
- Error Budget

**Resumo:**  
> SRE é quem garante que o sistema fique disponível e funcionando bem.

---

## 🎯 Diferença em uma frase

- **DevOps** → Cultura de entrega rápida.
- **SRE** → Engenharia focada em estabilidade.
